scapy基本介绍

在scapy中，每一个协议就是一个类。只需要实例化一个协议类，就可以创建一个该协议的数据包

scapy的协议类

Ether(),IP(),TCP(),UDP()

scapy基本操作

查看协议类的属性

ls()

ls(Ether())

设置协议类的属性

IP(src="192.168.1.1",dst="192.168.1.2",ttl=64) #设置ttl值，ttl值是一个数字

构造dst属性的值为1个IP地址的IP数据包

IP(src="192.168.1.1",dst="192.168.1.2")
dst的值是一个IP地址
生成了1个数据包

构造dst属性的值为1个IP范围的IP数据包

IP(src="192.168.1.1",dst="192.168.1.0/24")
dst的值是一个IP范围
生成了256个数据包

要查看其中的每一个数据包

[p for p in ip]

scapy数据包构造形式

scapy采用分层的形式来构造数据包，通常最下面的一个协议为Ether,然后是IP，再之后是TCP或UDP

生成一个广播数据包

Ether(dst="ff:ff:ff:ff:ff:ff")

scapy中的分层通过符号“/”实现，一个数据包是由多层协议组合而成，这些协议之间使用“/”分开，按照协议由底而上的顺序从左向右排列

构造一个TCP数据包

Ether()/IP()/TCP()

构造一个HTTP数据包

IP()/TCP()/"GET / HTTP/1.0\r\n\r\n"

scapy模块中的函数

send() #工作在第三层，用来发送IP数据包
sendp() #工作在第二层，用来发送Ether数据包

注：send()和sendp()只发不收，只会将数据包发送出去，但不能处理该数据包的回应包

fuzz() #构造一个内容是随机填充的数据包，又保证这个数据包的正确性
例：
IP(dst="192.168.1.1")/fuzz(TCP())
sr() #工作在第三层，用来发送和接收数据包
sr1() #工作在第三层，用来发送和接收数据包
srp() #工作在第二层，用来发送和接收数据包

sr()的返回值是两个列表，第一个列表是收到了应答的数据包和对应的应答数据包，第二个列表是未收到应答的数据包
所以可以使用两个列表来保存sr()的返回值

ans,unans=sr(IP(dst="192.168.1.1")/ICMP())

查看ans,unans列表中数据包的内容

ans.summary()
unans.summary()

判断数据包是否具有某协议

haslayer() 
例：resp.haslayer(TCP)

读取数据包中某协议字段的内容

getlayer()
例：resp.getlayer(TCP).flags

自动生成一个随机端口号

RandShort()

sr1()跟sr()作用基本一样，但是只返回一个应答数据包,所以只需使用一个列表来保存这个函数的返回值

p=sr1(IP(dst='192.168.1.1')/ICMP())

使用sr1()来测试目标的某个端口是否开放，采用半开扫描(SYN)方法：

p=sr1(IP(dst="192.168.1.1")/TCP(dport=80,flags="S"))
sniff() #捕获经过本机网卡的数据包

使用sniff()开始监听，但是捕获的数据包不会即时显示，只有使用ctrl+c停止监听时才会显示捕获的数据包
sniff()可以使用filter参数对数据包进行过滤，例：
只捕获与主机192.168.1.1有关的数据包

sniff(filter="host 192.168.1.1")

只捕获指定协议的数据包

sniff(filter="icmp")

如果要同时满足多个条件可以使用“and”，”or”等关系运算符来表达

sniff(filter="host 192.168.1.1 and icmp")

sniff()可以使用iface参数指定所要进行监听的网卡

sniff(iface="eth0")

sniff()可以使用count参数指定监听到数据包的数量，达到指定的数量就会停止监听

sniff(count=5)

sniff()参数组合使用

sniff(count=3,iface="eth0",filter="host 192.168.1.1 and icmp")

注：在scapy中，“_”下划线符号表示上一条语句执行的结果

a.summary() #以摘要的形式显示a的内容，摘要的长度为一行
a.nsummary() #作用与summary()相同，只是要操作的对象是多个数据包
a.show() #显示数据包的详细内容

scapy模块的常用简单实例

ACK端口扫描，探测目标端口是否被防火墙等过滤

from scapy.all import IP,TCP,sr
ans,unans=sr(IP(dst="192.168.1.1")/TCP(dport=[21,80,443,445],flags="A"))
for request,response in ans:
	if request[TCP].dport == response[TCP].sport:
		print(str(request[TCP].dport)," is unfiltered")
for req in unans:
	print(str(req[TCP].dport)," is filtered")

SYN端口扫描，探测目标端口开放或关闭

from scapy.all import IP,TCP,sr,fuzz
ans,unans=sr(IP(dst="192.168.1.1")/fuzz(TCP(dport=[21,22,80,443],flags="S")))
for request,response in ans:
	if response[TCP].flags == 18:
		print(str(response[TCP].sport)," is open")
	if response[TCP].flags == 20:
		print(str(response[TCP].sport)," is closed")
