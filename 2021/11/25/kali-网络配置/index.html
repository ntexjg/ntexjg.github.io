txjg blogkali 网络配置 2021-11-25

临时设置

ifconfig eth0 192.168.1.1 netmask 255.255.255.0 #IP,子网掩码设置
route add default gw 192.168.1.254 #默认网关设置
echo nameserver 114.114.114.114 > /etc/resolv.conf #永久设置dns服务器

注：这样设置的话，系统重启后IP，网关设置就会失效

写入配置文件

vi /etc/network/interfaces #打开网卡配置文件

添加

auto eth0
iface eth0 inet dhcp
address 192.168.1.1
netmask 255.255.255.0
network 192.168.1.0
broadcast 192.168.1.255
gateway 192.168.1.254

然后重启网络即可

/etc/init.d/networking restart
