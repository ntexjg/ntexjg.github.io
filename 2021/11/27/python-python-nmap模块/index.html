python-nmap模块使用前提

系统必须安装nmap

python-nmap模块安装

pip install python-nmap

python导入python-nmap模块

import nmap

python-nmap模块基本用法

  • python-nmap模块的核心类
    PortScanner，PortScannerAsync,PortScannerError,PortScannerHostDict,PortScannerYield
    其中最为重要的是PortScanner类
  • python-nmap模块类的实例化
    nm=nmap.PortScanner() #PortScanner类的实例化
nm=nmap.PortScannerAsync() #PortScannerAsync(与PortScanner功能类似，但可以实现异步扫描）实例化

python-nmap模块中的函数

PortScanner类

scan(hosts="127.0.0.1",ports="1-1000",arguments="-sV",sudo=False)

用来对指定目标进行扫描
hosts是要扫描的主机，可以是IP地址，也可以是域名
ports是要扫描的端口，单一端口：80 多个端口：80,443,3389 端口范围：1-1000
arguments是nmap扫描时所使用的参数
默认sudo=False,如果扫描需要sudo权限，则需要sudo=True

all_hosts() #返回一个被扫描的所有主机列表
command_line() #返回在当前扫描中使用的命令行命令
csv() #返回值是一个csv格式的扫描结果的输出
如果看不清楚，可以使用print(nm.csv())
has_host('127.0.0.1') #检查是否有该host的扫描结果，有为True,没有为False
scaninfo() #列出一个扫描信息的结构

这个类还支持以下操作

nm['192.168.1.1'].hostname() #获取192.168.1.1的主机名，通常为用户记录
nm['192.168.1.1'].state() #获取主机192.168.1.1的状态
nm['192.168.1.1'].all_protocols() #获取执行的协议
nm['192.168.1.1']['tcp'].keys() #获取TCP所有的端口号
nm['192.168.1.1'].all_tcp() #获取TCP所有的端口号(按照端口号大小进行排序）
nm['192.168.1.1'].all_udp() #获取UDP所有的端口号(按照端口号大小进行排序)
nm['192.168.1.1'].all_sctp() #获取SCTP所有的端口号(按照端口号大小进行排序)
nm['192.168.1.1'].has_tcp(22) #主机192.168.1.1是否有关于22端口的任何信息
nm['192.168.1.1']['tcp'][22] #获取主机192.168.1.1关于22端口的信息
nm['192.168.1.1'].tcp(22) #获取主机192.168.1.1关于22端口的信息
nm['192.168.1.1']['tcp'][22]['state'] #获取主机22端口的状态

PortScannerAsync类

scan(hosts="127.0.0.1",ports="80,443",arguments="-sV",callback=None,sudo=False)

相比PortScanner类多了一个回调函数callback,callback参数的值是一个函数,如：callback=callback_result,callback_result是函数名，函数是以(host,scan_data)为参数的函数
这个类提供了以下三个用来实现异步的函数

still_scanning() #如果扫描正在进行，则返回True,否则返回False
wait(2) #表示等待时间
stop() #停止当前的扫描
