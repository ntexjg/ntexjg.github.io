Medusa

查看可使用的模块

medusa -d

显示指定模块的帮助信息

medusa -M 模块 -q

Medusa常用参数

-h	目标主机名或IP地址
-H	包含目标主机名或IP地址的文件
-u	用户名
-U	用户名文件
-p	密码
-P	密码文件
-C	组合条目文件,格式为host:user:password
-O	日志信息文件
-e	n:空密码探测 s:密码等同用户名探测 ns:空密码和密码等同用户名探测
-M	要执行的模块名(没有.mod扩展名)
-m	传递给模块的参数.(可传递多个参数，如:-m para1 -m para2)
-d	显示所有的模块名称
-n	使用非默认端口
-s	启用SSL
-r	重试间隔时间，默认为3s
-t	设置线程数量
-L	并行化，每个用户使用一个线程
-f	在发现第一个有效的用户名/密码后停止破解
-q	显示模块的使用信息
-v	详细级别(0-6,默认为5)
-w	错误调试级别(0-10,默认为5)
-V	显示版本
-Z	继续扫描上一次

medusa破解实例

mysql破解

medusa -h 192.168.1.1 -u root -P pass.txt -M mysql

smbnt破解

medusa -h 192.168.1.1 -U user.txt -P pass.txt -e ns -M smbnt

mssql破解

medusa -h 192.168.1.1 -u sa -P pass.txt -t 5 -f -M mssql

ftp破解

medusa -C comb.txt -M ftp
注:comb.txt内容为
192.168.1.1:user:pass
192.168.1.2:user:
192.168.1.1::passwd

ssh破解

medusa -M ssh -H host.txt -U user.txt -p passwd

输出日志信息文件

medusa -h 192.168.1.1 -u root -p password -t 5 -f -M mysql -O log.txt
